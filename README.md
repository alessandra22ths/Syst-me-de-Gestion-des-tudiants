# Syst-me-de-Gestion-des-tudiants
Application conçue pour organiser facilement les informations des étudiants. Elle permet d’ajouter de nouveaux étudiants avec leur nom et âge, de consulter la liste des étudiants avec leur moyenne, d’enregistrer des notes pour chaque étudiant, de modifier ou supprimer des profils, de rechercher un étudiant par nom, de visualiser le classement selon les moyennes, et d’exporter toutes les données dans un fichier CSV pour analyses ou rapports. Les informations sont automatiquement sauvegardées dans un fichier texte pour conserver toutes les données entre les sessions. Cette application est idéale pour pratiquer la programmation, manipuler des listes et dictionnaires, gérer des fichiers et créer un outil fonctionnel à montrer dans son portfolio.
